# Auto-Boost-Av1an-portable
Fully automated AV1 encoding workflow for Windows. Drop .mkv files, double-click a batch .bat, done. Uses av1an with Auto-Boost for consistent visual-metric quality. Supports any quarter-step CRF SVT-AV1 fork, zones.txt, and multiple worker backends. Handles auto-renaming, encoding, and muxing original audio/subs.
